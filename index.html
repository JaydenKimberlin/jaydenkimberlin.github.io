Guessing Game

This game is just about guessing. You guess a number 1-100, and bada bing bada boom, you get the wrong answer because that is a retarded range. It's really really just self explanatory, if you don't know what this is, then just get outta here. If one wasn’t from this class, there’s no explanation needed. You hit the “start game” button and then just play. The first version only took a few hours as it was rushed, even with the help of a fellow classmate. The second iteration took about an hour and a half while working simultaneously with another classmate, all rewritten into the guessing game it is now.
